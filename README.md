# ChatGPT Canvas Clone  

เป็นโปรเจ็คสำหรับ clone chat gpt canvas โดยที่หลักๆจะเน้นฝึกไปทางฟังชันจำลองเขียนเอกสารบนเว็ปแอพและในอนาคตมีการจะเชื่อมต่อร่วมกับ AI Agent สำหรับช่วยเขียนเอกสาร

## npm Libraries Used  

- `docxtemplater`  
- `draft-js`  
- `pdfmake`  
- `markdown-draft-js`
